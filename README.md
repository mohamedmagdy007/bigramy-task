# React + Vite

### Prerequisites

- Node.js ( v20.11.0)

### Installation

1. Clone the repository:

```sh
git clone https://github.com/mohamedmagdy007/bigramy-task.git
```

2. Install dependencies:
   ```sh
   npm install
   ```

## Development

To start the development server, run:

```sh
npm run dev
```
